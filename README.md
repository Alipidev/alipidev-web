# Alipidev Web Project

## Descripción
El proyecto Alipidev es un sitio web de portafolio diseñado para Alina, una desarrolladora web junior en el ámbito Full Stack. Este sitio exhibe las habilidades y proyectos de Alina, proporcionando una plataforma donde potenciales empleadores y clientes pueden explorar su trabajo y experiencias.

## Características
- Diseño responsivo y atractivo
- Navegación sencilla e intuitiva
- Sección de proyectos destacando los trabajos más relevantes
- Información sobre servicios ofrecidos

## Servicios
- Desarrollo de aplicaciones web
- Diseño de interfaces de usuario
- Consultoría técnica

## Habilidades Técnicas
- HTML, CSS, JavaScript
- Frameworks como React y Node.js
- Familiaridad con bases de datos como MongoDB y MySQL

## Estructura del Proyecto
- `index.html`: Página de inicio
- `proyectos.html`: Sección de proyectos
- `servicios.html`: Información sobre servicios
- `contacto.html`: Formulario de contacto

## Tecnologías Utilizadas
- HTML5 y CSS3 para la estructura y estilo
- JavaScript para interactividad
- Frameworks modernos para un desarrollo eficiente
